Code sample of a Bootstrap3 layout with Font-Awesome icons and AJAX updates provided via AngularJS.

This first page utilizes responsive functionality from Bootstrap to provide an engaging experience for mobile and web users regardless of screen size and ratio.

The background uses personal photos, with a parallax effect added to mirror the immersive effect of going under the surface of the bay.

The TideChart GUI showcases AJAX updates of SVG graphics via AngularJS calls to a SpringBoot RESTful Java8 API with an H2, in-memory database initialized upon application startup.

Plaaned future expansion of the TideChart GUI includes location awareness with availability to select and pull from local tide buoy.
